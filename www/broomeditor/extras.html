Embedded Python shell, with default loading of broom connection to specified database.
Table listing output of lint messages, updated in real-time.
  • Flag (notice, warning, error)
  • Line number
  • Message
Table of open review requests, with ability to open existing review, change filter to include other statuses, request a review, etc. Fields include review ID, files included, date opened, status, requester, target review group.
Table of open issues (linked to ticketing system, e.g. JIRA), with ability to view, sort, close, etc., and also open an external page with the review.
Listing of unit tests linked to from open file(s). Shows if test has run since last save/change, and lists run status, etc. Unit tests determined by looking at header of source file, e.g., YAML lines with some metadata.
Debugging interface, with watched variables, current stacktrace, and debugging options (add breakpoint, etc.). Active when a script it run in the interpreter at all times, or enabled via setting of at least one breakpoint in the editor.
